Para instalarlo: 
	npm install en backend
	npm install en frontend

	instalar mongodb (sin usuario ni contraseña)
	mongod --dbpath="%ruta de la carpeta"/backend/data"

El dump de la bdd se guarda en backend/data
Hay usuarios de prueba (cosa@cosa.cosa  con contraseña "cosa" y test@test.es con contraseña "test"), pero se pueden crear usuarios.
